## Infrastructure for Modern Auth

OTP Edge is a WhatsApp OTP platform designed for developers who prioritize security, speed, and reliability. Connect your Meta WhatsApp Business credentials once in the dashboard, then use a simple API to dispatch OTPs and verify users.

### Key Features

  <Card

## The OTP Edge Flow

```javascript
// 1. Dispatch
await fetch('https://otpedge.com/api/v1/send-otp', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ 
    to: "+919876543210", 
    app: "MyAwesomeApp" 
  })
});

// 2. Verify
const response = await fetch('https://otpedge.com/api/v1/verify-otp', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ 
    phone_number: "+919876543210", 
    code: "123456" 
  })
});

// 3. Receive Stateless Identity
// Returns: { identity_token: "eyJhbG..." }
// See "Verifying Identity Tokens" for backend validation.
```
