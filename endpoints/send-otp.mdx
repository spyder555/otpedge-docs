---
title: 'Send OTP'
api: 'POST https://api.otpedge.com/v1/send-otp'
description: 'Dispatches a 6-digit verification code to a WhatsApp number.'
---

## Request Body

<ParamField body="to" type="string" required>
  The recipient's phone number in E.164 format (e.g., `919876543210`).
</ParamField>

<ParamField body="app" type="string" required>
  The name of your application, used for branding in the message.
</ParamField>

<ParamField body="template" type="string" optional>
  The name of the WhatsApp template to use. If omitted, the system will use the most recent template registered for the workspace, defaulting to `verification_otp`.
</ParamField>

<ParamField body="language" type="string" optional>
  Language code for the template (e.g., `en_US`). Defaults to `en_US` if not provided.
</ParamField>

<ParamField body="environment" type="string" optional>
  Set to `sandbox` or `test` to receive the generated OTP in the response for debugging. Defaults to `live`.
</ParamField>

```go Go (net/http)
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://api.otpedge.com/v1/send-otp"
	payload := map[string]string{
		"to":  "919876543210",
		"app": "MySecureApp",
		"template": "my_custom_template",
		"language": "en_US",
		"environment": "sandbox",
	}
	jsonPayload, _ := json.Marshal(payload)

	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonPayload))
	req.Header.Set("Authorization", "Bearer your_api_key_here")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{}
	res, err := client.Do(req)
	if err != nil {
		fmt.Println(err)
		return
	}
	defer res.Body.Close()

	fmt.Println("Response Status:", res.Status)
}
```
